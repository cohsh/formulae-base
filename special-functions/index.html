Special Functions

Gamma Function

$$\Gamma(z) := \int_0^\infty dt t^{z-1}e^{-t},~\mathrm{Re}(z) > 0$$
$$\Gamma(z) := \lim_{n\ o \infty} \ rac{n^{z}n!}{\prod_{j=0}^n(z+j)}$$
$$\Gamma(z+1) = z\Gamma(z),~ \mathrm{Re}(z) > 0$$

Euler's Reflection Formula

$$\Gamma(z)\Gamma(1-z) = \frac{\pi}{\sin{\pi z}},~ z \notin \mathbb{Z}$$

Legendre Duplication Formula

$$\Gamma(z)\Gamma(z+\ rac12) = 2^{1-2z}\pi^\ rac12 \Gamma(2z)$$

Beta Function

$$\mathrm{B}(z_1, z_2) := \int_0^1 dt t^{z_1-1}(1-t)^{z_2-1},~\mathrm{Re}(z_1),\mathrm{Re}(z_2)>0$$
$$\mathrm{B}(z_1, z_2) := 2\int_0^{\ rac{\pi}{2}} d\ heta \sin^{2z_1-1}\ heta \cos^{2z_2-1} \ heta,~\mathrm{Re}(z_1),\mathrm{Re}(z_2)>0$$
$$\mathrm{B}(z_1, z_2) = \ rac{\Gamma(z_1)\Gamma(z_2)}{\Gamma(z_1 + z_2)}$$
$$\mathrm{B}(z_1, z_2) = \mathrm{B}(z_2, z_1)$$
$$\mathrm{B}(z_1, z_2) = \mathrm{B}(z_1, z_2 + 1) + \mathrm{B}(z_1 + 1, z_2)$$
$$\mathrm{B}(1, z_1) = \ rac{1}{z_1}$$

Riemann Zeta Function

$$\zeta(z) = \sum_{n=1}^\infty \ rac{1}{n^z}$$

Error Function

$$\mathrm{erf}(z) := \ rac{2}{\pi^{\ rac12}} \int_0^z dt e^{-t^2}$$
$$\mathrm{erf}(-z) = -\mathrm{erf}(z)$$
$$\mathrm{erf}(z^*) = \mathrm{erf}(z)^*$$
