Kohn-Sham Equation

$$\left[ -\frac{1}{2} \nabla^2 + V_\mathrm{en}(\mathbf{r}) + V_\mathrm{Hxc}[n](\mathbf{r}) \right] \psi_i[n](\mathbf{r}) =\epsilon_i[n] \psi_i[n](\mathbf{r})$$
$$n(\mathbf{r}) := \sum_i \left|\psi_i[n](\mathbf{r})\right|^2$$
$$V_\mathrm{en}(\mathbf{r}) := -\sum_i Z_i v(\mathbf{r} - \mathbf{R}_i)$$
$$V_\mathrm{Hxc}[n](\mathbf{r}) := \frac{\delta E_\mathrm{Hxc}[n]}{\delta n}$$
$$E_\mathrm{Hxc}[n] := E_\mathrm{H}[n] + E_\mathrm{xc}[n]$$
$$E_\mathrm{H}[n] := \frac12 \int d\mathbf{r}_1 d\mathbf{r}_2 n(\mathbf{r}_1)v(\mathbf{r}_1 - \mathbf{r}_2) n(\mathbf{r}_2)$$
